НИЯУ МИФИ

Низколатентная система визуального позиционирования

Задача: разработать аппаратную систему, способную в реальном времени детектировать и отслеживать динамические цели на экране и управлять 2D-платформой на кинематике H-bot для точной корректировки движения компьютерной мыши. Тестовая среда — Aimlabs, CS2, Valorant

Ключевые особенности:

  • Аппаратная изоляция: любое программное взаимодействие с хост-компьютером исключено
  • CAD: SolidWorks
  • ПО: C++, Python, OpenCV, Git, VScode
  • Электроника: RPi 5, RPi camera, STM32, Nema 17, TMC2209
  • Производство: Лазерная резка (алюминий, сталь), 3D-печать

Крупноформатный лазерный ЧПУ

Задача: Спроектировать крупноформатный (800x800 мм) лазерный станок

Ключевые особенности:

  • Кастомные узлы: все ключевые компоненты (каретки, корпус для электроники и т.д.) спроектированы в Компас-3D и изготовлены методом FDM-печати для адаптации конструкции под выбранные комплектующие.
  • Активная доработка: ведется проектирование защитного корпуса со встроенной вытяжкой для повышения безопасности и расширения спектра обрабатываемых материалов.
  • CAD: Компас-3D (проектирование), SolidWorks (рендер)
  • ПО: LightBurn, LaserGRBL
  • Электроника: MKS DLC 32, Nema 17, A4988
  • Производство: 3D-печать, слесарная сборка

Робот-погрузчик

Задача: Создать миниатюрного робота-погрузчика для инженерных игр, способного распознавать цвет грузов, осуществлять их захват, перевозку и подъем на заданную высоту для сброса

Ключевые особенности:

  • CAD: Компас-3D
  • ПО: MicroPython, VScode, Arduino IDE

  • Производство: 3D-печать
