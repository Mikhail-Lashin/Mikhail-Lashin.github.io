Аппаратный аимбот для fps-шутеров

(Статус: в процессе)

Задача: разработать невидимое для клиентского античита устройство, способное в реальном времени детектировать и отслеживать динамические цели на экране (в Aimlabs, CS2, Valorant) и управлять платформой на кинематике H-bot для корректировки движений компьютерной мыши

Ключевые особенности:

  • Аппаратная изоляция: любое программное взаимодействие с хост-компьютером исключено
  • Соотношение жесткость-инерция: подвижные элементы облегчены методом топологической оптимизации
  • CAD: SolidWorks
  • ПО: C++, Python, OpenCV, Git, VScode
  • Электроника: RPi 5, RPi camera, Nema 17, TMC2209
  • Производство: лазерная резка (алюминий, сталь), 3D-печать

Крупноформатный лазерный ЧПУ

(Статус: завершено*)

Задача: спроектировать с нуля и собрать крупноформатный (800x800 мм) лазерный станок

Ключевые особенности:

  • CAD: Компас-3D (проектирование), SolidWorks (рендер)
  • ПО: LightBurn, LaserGRBL
  • Электроника: Laser Module 10W 455nm, MKS DLC 32, Nema 17, A4988
  • Производство: 3D-печать, слесарная сборка

*В дополнение к изначальной задаче ведется проектирование защитного корпуса со встроенной вытяжкой для повышения безопасности и расширения спектра обрабатываемых материалов (акрил, EVA и т.п.)

Вал приводной

(Статус: завершено)

Задача: по предоставленному сборочному чертежу советского образца выполнить реверс-инжиниринг приводного вала (спроектировать сборку из 42 деталей и оформить частичный набор КД с указанием допусков и посадок)

Ключевые особенности:

  • CAD: Компас-3D
  • Дополнительно: библиотека "валы и механические передачи"
  • Литература: справочники Анурьева, Анухина, Курмаза

Робот-погрузчик

(Статус: завершено)

Задача: создать миниатюрного робота-погрузчика для инженерных игр, способного распознавать цвет грузов, осуществлять их захват, перевозку и подъем на заданную высоту для сброса.

Ключевые особенности:

  • CAD: Компас-3D
  • ПО: MicroPython, VScode, Arduino IDE
  • Электроника: ESP32, color sensor RGB, servo MG90S
  • Производство: 3D-печать
